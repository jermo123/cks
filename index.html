Whipped Topping Guide

Perfect Consistency

  • Should hold firm peaks
  • Not too soft (will collapse)
  • Not too stiff (will be rough)
  • Temperature: Keep below 40°F

Common Issues

  • Melting: Return to cooler for 5-10 min
  • Bubbles: Gently tap piping bag
  • Too Stiff: Let warm slightly
  • Too Soft: Return to cooler

Airbrush Guide

Basic Techniques

  • Distance: 6-8 inches for even coverage
  • Pressure: Medium (test on paper first)
  • Motion: Circular for smooth blend
  • Clean between colors thoroughly

Color Effects

  • Pearl + Pink = Shimmer Rose
  • Blue + Pearl = Ice Effect
  • Gold + Pearl = Metallic Shine
  • Multiple Colors = Rainbow
  • Light spray of Pearl = Sparkle Effect

Common Issues

  • Splattering: Increase distance
  • Uneven color: Overlap strokes
  • Too dark: Reduce pressure
  • Drips: Move faster, less color

Special Effects

  • Stencils: Hold firmly, light spray
  • Fading: Gradually increase distance
  • Patterns: Use template guides
  • Borders: Use cake strip as guide

Getting Started with Airbrush

Setup
  • Clean workspace required
  • Cover surrounding areas
  • Keep paper towels handy
  • Wear gloves
Basic Technique
  • Hold 6-8 inches away
  • Use consistent pressure
  • Keep moving - don't stop
  • Practice on paper first
Common Mistakes
  • Too close = splatters
  • Too much color = runs
  • Stopping = spots
  • Rushing = uneven coverage
Cleaning Tips
  • Clean between colors
  • Flush with warm water
  • Test spray after cleaning
  • Keep nozzle clean

Edible Image Printer Guide

Daily Maintenance

  • Run nozzle check before first print
  • Keep printer covered when not in use
  • Check humidity levels (40-60% ideal)
  • Clean paper sensor with dry cloth

Troubleshooting

  • Streaky prints:
    • Run head cleaning
    • Check for expired cartridges
    • Verify humidity levels
  • Paper jams:
    • Remove paper carefully
    • Clean rollers with damp cloth
    • Let rollers dry completely

Tips & Tricks

  • Store sheets in sealed bag
  • Handle sheets by edges only
  • Print within 15 mins of opening
  • Use sheets at room temperature
  • Save ink: draft mode for tests

Cleaning Process

  1. Power off printer
  2. Mix 1:1 water/alcohol solution
  3. Dampen lint-free cloth slightly
  4. Clean rollers in one direction
  5. Wait 30 mins before use

Never use cleaning solutions inside print head area!
